## Brief description of the codebase

As instructed, the files are located inside the client folder.

The project is based on the stable version of Nuxt (v4.3.1)

## Pages / Routing:
I followed the usual way of routing the site. It consists of only the index and the game detail page.

## Components:
I made a component for the game item itself for the list so that it will be reusable instead of creating it in the index file.

I also added a component for the header file so that I can add the header text and the back button at the top.
